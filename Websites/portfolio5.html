HTML Programming

What is the structure of an HTML document?

An HTML document begins with the **DOCTYPE** declaration, followed by the **<html>** tag that contains the entire page. Inside it, the **<head>** section holds metadata and the **<title>**, while the **<body>** section contains all the visible content. This structure ensures that browsers understand how to display the webpage correctly.

How are HTML tags used to structure content on a webpage?

HTML tags define different parts of a webpage such as **headings**, **paragraphs**, **images**, **links**, and **tables**. Each tag has a specific purpose and helps organize content so it is readable and structured. These tags guide the browser on how to display each section of the webpage.

What are common HTML tags?

Common HTML tags include **<h1>** for headings, **<p>** for paragraphs, **<a>** for links, and **<img>** for images. Other important tags include **<div>** for layout, **<ul>** and **<li>** for lists, and **<table>** for tabular data. These tags create the foundation of most web pages.

CSS Programming

How does CSS enhance HTML?

**CSS** enhances HTML by adding styling such as **colors**, **spacing**, **fonts**, **borders**, and **layouts**. Without CSS, web pages would appear plain and unformatted. With CSS, developers can create visually appealing and user-friendly designs.

What are the main components of CSS syntax?

CSS syntax consists of a **selector**, **properties**, and **values**. The **selector** chooses the HTML element, the **property** defines what to change (like color or font size), and the **value** tells how to apply that change. Together, these components allow precise control over webpage appearance.

How can CSS be applied to improve the look and feel of web pages?

CSS can be applied **inline**, inside the HTML document, or through an **external stylesheet**. It improves the look and feel by controlling **layout**, **aligning elements**, and creating **consistent styles** across multiple pages. CSS also supports animations, responsive design, and customized themes.

Future of Computing: Generative AI and Data Analytics

What is Generative AI, and how does it differ from traditional AI?

**Generative AI** creates **new content** such as text, images, and music, while **traditional AI** mainly **analyzes data** and makes predictions. This makes generative AI more creative and capable of producing original outputs. Traditional AI focuses on classification and decision-making, while generative AI expands into creativity.

How are data analytics transforming industries?

**Data analytics** helps industries make **smarter decisions** by analyzing large datasets to uncover **patterns and trends**. It is used in finance for fraud detection, in healthcare for diagnosis, and in business for predicting customer behavior. These insights lead to better strategies, reduced costs, and improved services.

What ethical considerations are associated with AI?

AI raises concerns about **data privacy**, **misinformation**, **job displacement**, and **biases in algorithms**. These issues require responsible development and regulation to prevent harm. Ethical AI ensures **fairness**, **transparency**, and **accountability**.
